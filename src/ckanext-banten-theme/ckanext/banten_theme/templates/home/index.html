Kenapa Satu Data Banten?

Kami menjaga data tetap rapi, terstruktur, dan dapat dipercaya.

Data Terstruktur

Seluruh dataset mengikuti standar SDI dan format terbuka (CSV, JSON, XLSX) sesuai amanat Perpres No. 39/2019.

Verifikasi OPD

Tiap dataset bisa dipantau jejak datanya, dari OPD penyedia di Provinsi Banten hingga riwayat pembaruannya.

API Terbuka

CKAN menyediakan REST API standar untuk integrasi aplikasi pihak ketiga, riset, dan jurnalisme data.
